# ABA 篮球计分牌

ABA 联赛风格篮球比赛计分牌工具，适用于业余篮球赛事现场计分展示。

## 在线体验

GitHub Pages: `https://yefei404.github.io/basketball_scoreboard/`

## 功能特性

- **比分控制**：+1 / -1 按钮快速调分，支持 contenteditable 手动修改
- **倒计时器**：每节 12 分钟倒计时，支持开始/暂停/重置
- **节次显示**：可编辑的节次标识（Q1、Q2 等），直接点击修改
- **暂停管理**：每队 5 次暂停，支持使用和加回，以横线可视化展示
- **队伍自定义**：队名可编辑，队伍色块颜色可自定义
- **全面可定制**：顶部色块、底部比分框底色均可通过颜色选择器调整
- **iPad 桌面图标**：保存到主屏幕时显示奖杯图标

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 点击队名直接编辑（如"浙江阿勒克斯"→ 你的队名）
3. 点击 Q1 可修改为 Q2、Q3、Q4 等
4. 使用控制面板操作计时器、暂停和比分
5. 通过颜色选择器自定义各区域颜色

## 设计说明

### 界面布局

```
┌─────────────────────────────┐
│     Q1      │    12:00      │  ← 顶部行：节次 + 倒计时
├─────────────┼───────────────┤
│             │               │
│  浙江阿勒克斯 │   宁夏太平    │  ← 中间行：队伍名称（可编辑）
│   (红底白字)  │  (蓝底白字)   │
├─────────────┼───────────────┤
│  - - - - -  │  - - - - -    │  ← 暂停行：5 次暂停标记
├──────┼──────┼──────┼────────┤
       │  0   │   0  │           ← 底部行：比分（居中 50% 宽度）
       └──────┴──────┘
```

### 技术栈

| 层面 | 方案 |
|------|------|
| 结构 | 原生 HTML5 |
| 样式 | 原生 CSS（内联） |
| 逻辑 | 原生 JavaScript |
| 部署 | GitHub Pages |

### 计时器

- **模式**：倒计时（12:00 → 0:00）
- **精度**：setInterval 1 秒间隔
- **到时行为**：自动停止，按钮恢复为"开始"

### 暂停系统

- 每队初始 5 次暂停
- 使用暂停：减少一个横线标记
- 加回暂停：恢复一个横线标记（用于误操作撤销）
- 上限 5 次，下限 0 次

### 颜色自定义

| 区域 | 默认颜色 | 说明 |
|------|----------|------|
| 左队背景 | `#ff0000` 红色 | 队伍名称色块 |
| 右队背景 | `#00bfff` 天蓝 | 队伍名称色块 |
| 顶部色块 | `#5b72b5` 蓝灰 | 节次和计时器背景 |
| 比分框底色 | `#ffffff` 白色 | 底部比分区域 |

### contenteditable 可编辑区域

- 节次（Q1）
- 左队名称
- 右队名称
- 左队比分
- 右队比分
